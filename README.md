# membery

반려동물 800만 시대! 자신의 반려동물을 자랑하고 
여행이나 휴가 때 안심하고 맡길 수 있는 반려동물 펫시터를 구할 수 있는 웹페이지

----------------
### 주요 기능
+ 로그인
  + 회원가입
  + 로그인
  + 소셜 로그인
  + 회원탈퇴
+ sns기능
  + 마이피드
  + 반려동물 등록
  + 홈 피드
+ 펫시터
  + 메인페이지
  + 전체보기
  + 호스트 등록
  + 상세페이지 등록
+ 캘린터
+ 함께가개
  + 반려동물 동반 가능 식당/카페/숙소 검색
  + 원하는 장소 찜
